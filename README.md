# EduHub – Thế giới cổ tích

Nền tảng học tập trực tuyến dành cho học sinh và giáo viên.

## Tech Stack

- **Frontend:** Vite + React + TypeScript + Tailwind CSS
- **Backend:** FastAPI + SQLite

## Cấu trúc dự án

```
EduHub/
├── frontend/   # Vite + Tailwind
├── backend/    # FastAPI
```

## Chạy dự án

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
